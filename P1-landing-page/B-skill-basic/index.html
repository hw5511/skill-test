What You'll Learn

이 워크샵에서 배우는 것

AI 에이전트 개발에 필요한 핵심 역량을 4가지 영역으로 압축했습니다.

🧠

에이전트 설계 원리

목표 분해, 계획 수립, 도구 선택 등 자율 에이전트의 내부 작동 원리를 이해하고 직접 설계합니다.

🔧

도구 연동 & MCP

파일 시스템, 웹 검색, 외부 API를 연결하는 MCP 서버를 구성하고 에이전트에 붙이는 방법을 익힙니다.

⚡

워크플로우 자동화

반복 업무를 에이전트에 위임하는 자동화 파이프라인을 설계하고 실제 업무에 적용합니다.

🚀

실전 프로젝트 배포

팀 단위 프로젝트를 기획·개발·배포까지 완성하며 포트폴리오 수준의 결과물을 만들어냅니다.

Schedule

4주 커리큘럼

매주 토요일 오전 10시 ~ 오후 6시, 온·오프라인 병행 진행.

Week 1

에이전트 기초 & 환경 세팅

LLM API 연동, Claude Code 설치, 첫 번째 에이전트 실행. 에이전트와 일반 챗봇의 차이를 직접 체험합니다.

Claude API Claude Code 프롬프트 엔지니어링
Week 2

도구 활용 & MCP 연동

파일 편집, 웹 검색, Notion·슬랙 API를 에이전트에 연결합니다. MCP 서버를 직접 구성해봅니다.

MCP Server Tool Use API 연동
Week 3

자동화 파이프라인 설계

멀티스텝 태스크 분해, 에러 처리, 사람-에이전트 협업 구조를 설계하고 실무 시나리오에 적용합니다.

워크플로우 설계 Human-in-the-loop 에러 핸들링
Week 4

팀 프로젝트 & 데모

3~4인 팀으로 실제 문제를 해결하는 에이전트를 기획·개발·발표합니다. 수료증이 수여됩니다.

팀 프로젝트 배포 데모 발표
