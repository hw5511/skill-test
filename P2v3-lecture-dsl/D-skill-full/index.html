INTRODUCTION

이 교육에서 배우는 것

이 과정은 AI 에이전트의 기초부터 실전 활용까지를 다룹니다.

단순 챗봇과 달리 AI 에이전트는 목표를 설정하고, 계획을 세우고, 도구를 활용하여 스스로 작업을 수행하는 자율 시스템입니다.

4주간의 교육을 통해 여러분은: - AI 에이전트의 핵심 원리를 이해하고 - Claude Code를 활용한 실습을 경험하고 - 실제 업무에 적용 가능한 자동화 워크플로우를 설계하게 됩니다

CONCEPT

AI 에이전트의 4가지 핵심 능력

01
환경 인식(Perception)

파일 시스템, 웹 페이지, API 등 외부 데이터를 탐색하고 수집합니다

02
자율 판단(Reasoning)

수집한 정보를 분석하여 최적의 행동을 결정합니다

03
도구 활용(Tool Use)

코드 실행, 파일 편집, 터미널 명령 등 실제 작업을 수행합니다

04
반복 학습(Iteration)

결과를 확인하고 오류가 있으면 스스로 수정합니다

COMPARISON

일반 AI vs AI 에이전트

일반 AI (ChatGPT 등)

질문에 답변하는 1회성 도구입니다.

사용자가 매번 새로운 질문을 입력해야 하며, AI가 직접 파일을 수정하거나 코드를 실행할 수 없습니다.

한계: 복잡한 멀티스텝 작업 불가

AI 에이전트 (Claude Code 등)

목표를 설정하고 스스로 계획-실행-수정하는 자율 시스템입니다.

파일 읽기/쓰기, 터미널 명령, 웹 검색 등 도구를 자유롭게 활용하여 복잡한 작업을 자동 완료합니다.

강점: 100단계 이상의 복잡한 작업도 자율 수행
